# Anotações de HTML e CSS

## Truque para alterar o estilo em tempo real

```
<!DOCTYPE html>
<html>
    <head><title>Test</title></head>
    <body>
        <style contenteditable style="display:block; white-space: pre;">
            html {
                background-color: red;
            }
        </style>
    </body>
</html>
```
